# mthesis

### [Thesis(PDF) is here.](https://kmr-gks.github.io/mthesis/main.pdf)

### How to build the PDF:

1. Install texlive

Windows: [https://www.tug.org/texlive/windows.html](https://www.tug.org/texlive/windows.html)

Linux: [https://www.tug.org/texlive/quickinstall.html](https://www.tug.org/texlive/quickinstall.html)

macOS: [https://www.tug.org/mactex/](https://www.tug.org/mactex/)

2. Open VS Code and install the LaTeX Workshop extension.
3. Open tex_source/main.tex
